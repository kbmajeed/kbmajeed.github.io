Exposure Fusion

  • Python, OpenCV, Numpy

Exposure fusion is a technique that creates a single image with optimal detail from a set of multi-exposed images. As developed by Tom Mertens et al., the proposed algorithm computes relevant quality measures; Contrast, Saturation, and Well-Exposedness. These measures are then combined to create a weight map used to blend each of the multi-exposed images to a single image with best exposure.

SeaLion Classification

  • Matlab, Python, CNN

A computer vision algorithm is designed to find and counts sea lions from aerial images while also categorizing found sea lions to classes: adult males, sub adult males, adult females, juveniles and pup respectively. First, a high-resolution aerial image is cut into smaller images. Next, patches extracted from the smaller images are used to train a convolutional neural network for classification
